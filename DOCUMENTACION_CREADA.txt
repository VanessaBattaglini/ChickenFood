🎓 CONCEPTOS CLAVE
═════════════════════════════════════════════════════════════════════════════════

Intent
└─ Mecanismo para navegar entre actividades

Callback
└─ Función que se ejecuta cuando ocurre un evento (ej: click)

Estado local
└─ Variable que mantiene su valor dentro de un Composable

ViewModel
└─ Clase que mantiene datos y lógica de negocio

SQLite
└─ Base de datos local en el dispositivo

Firebase
└─ Base de datos en la nube

enableEdgeToEdge()
└─ Permite que la UI se extienda a los bordes

MaterialTheme
└─ Proporciona Material Design 3 theming
